Base macro for marking the start of a code block that runs at regular intervals.

Note
It is essential to use the RUN_END_ macro to conclude the code block.
DO NOT USE this macro directly in your application. Instead, utilize appropriate wrappers such as RUN_EVERY_US_OBJ_(), RUN_EVERY_US_OBJ_FORCE_(), RUN_EVERY_US_(), RUN_EVERY_US_FORCE_(), etc.
Parameters
name_Name of the code block.
interval_The time interval for executing the code block.
chrono_A pointer to the chrono object.
ts_The time sample of the RUN_EVERY object. Use Us for microseconds, Ms for milliseconds, and S for seconds.
force_If true, RUN_EVERY will execute the code block immediately; if false, it will only initialize its chrono object and run the code block in the next interval.
